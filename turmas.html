Nossas Turmas

Duração: 3 anos Período Integral Turma 2024

Curso Técnico

Formação técnica completa com foco em desenvolvimento profissional e preparação para o mercado de trabalho.

O curso técnico oferece uma formação sólida em diversas áreas técnicas, combinando teoria e prática para preparar os alunos para o mercado de trabalho.

Com duração de 3 anos, o curso inclui estágios supervisionados e projetos práticos que desenvolvem habilidades técnicas e profissionais.

Desenvolvimento de Jogos 2D

Nossa turma de Curso Técnico inclui um módulo especializado em desenvolvimento de jogos 2D, onde os alunos aprendem:

Ferramentas e Tecnologias:
  • Unity 2D
  • GameMaker Studio
  • Construct
  • Godot Engine
  • Adobe Photoshop
Conceitos Aprendidos:
  • Game Design
  • Programação C#
  • Arte Digital
  • Animação 2D
  • Sound Design
Sites e Recursos Recomendados:
  • Unity Learn - Tutoriais oficiais
  • GameDev.net - Comunidade de desenvolvedores
  • itch.io - Plataforma de jogos independentes
  • Brackeys - Canal de tutoriais
  • OpenGameArt - Recursos gratuitos
  • Gamasutra - Artigos sobre game design
Recursos Adicionais:
  • W3Schools - Aprenda programação web
  • freeCodeCamp - Cursos gratuitos de programação
  • Udemy - Cursos técnicos online
  • Coursera - Cursos universitários online
  • GitHub - Repositório de códigos
  • Stack Overflow - Comunidade de programadores

Propedêutico

Preparação intensiva para vestibulares e concursos públicos, com foco em todas as disciplinas fundamentais.

O curso propedêutico oferece uma preparação completa e estruturada para os principais vestibulares do país, incluindo ENEM, FUVEST, UNICAMP e outros concursos públicos.

Com metodologia comprovada e professores especializados, nossos alunos alcançam excelentes resultados nos exames de admissão.
